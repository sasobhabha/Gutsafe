GutSafety — deploy API + Netlify frontend
=========================================

Netlify (gutsafe.netlify.app) only hosts HTML/JS. The barcode API must run elsewhere.

A) Deploy FastAPI (recommended: Render)
---------------------------------------
1. Push this repo to GitHub/GitLab.
2. https://dashboard.render.com → New → Web Service → connect repo.
3. Runtime: Docker. Dockerfile path: Dockerfile (repo root).
4. Create Web Service. Wait for deploy; copy the URL, e.g. https://gutsafe-api.onrender.com

B) Point the website at the API
--------------------------------
1. Open web/api-config.js
2. Set: window.__GUTSAFE_API__ = "https://YOUR-RENDER-URL";
   (no trailing slash)
3. Commit, push; Netlify will redeploy.

Optional env on Render: USDA_FDC_API_KEY, SMARTLABEL_API_KEY, SMARTLABEL_CONFIGURATION_ID
